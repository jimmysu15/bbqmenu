為何製作此工具?

網頁上的菜單，因具有菜色照片預覽功能，在 HTML 中夾帶著一些額外的屬性或超連結，如遇到換季或需要大幅調整菜單的狀況，無法使用一般的所見即所得網頁工具加快處理流程。故製作此工具。

使用方法

1.將所有菜色的 html ，置於右側菜色區內。
2.將選定的菜色移至左邊菜單。
3.按下產生 HTML。
4.將產生的 HTML 貼回菜單網頁檔案內。
